Image Quality (IQA) Scorer

How to use

This page measures the perceptual quality of one or more distorted images relative to a single clean reference image, using a trained full-reference IQA model. Everything runs in your browser — no install, and no image is uploaded to any server.

  1. Load model — pick your exported .onnx model file.
  2. Choose reference image — the original / undistorted image (one only; a preview is shown).
  3. Choose distorted images — select one or many at once. These are the versions you want to evaluate (e.g. compressed, blurred, denoised, upscaled). No previews are shown; you'll see the filenames instead.
  4. Compute scores — the model scores each distorted image against the reference and fills in a results table.
  5. Export CSV — download all reference/distorted pairs and their scores.

Notes: each distorted image must depict the same content as the reference, or the score is meaningless. Whether a higher or lower score means "better" depends on how your model was trained — interpret it against your own validation set.
